# JavaScript-Bot-Base
- A template Discord Bot. Add your own commands by creating them in the commands folder! 

## Setup
* Rename `.env.example` to `.env`
* Fill in `.env` with the proper information
* `npm install` 

## Includes
* Command & Event Handler
* Ping Command
* Base Events
